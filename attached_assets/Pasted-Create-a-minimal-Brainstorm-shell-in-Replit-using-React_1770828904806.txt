Create a minimal Brainstorm shell in Replit using React + Vite + TypeScript.

This shell is designed for the backend team to integrate applesauce (https://github.com/hzrd149/applesauce) for Nostr functionality and connect to brainstormserver API.

STACK:
- React 18 + TypeScript + Vite
- Tailwind CSS (minimal)
- wouter for routing

STRUCTURE:
src/
├── pages/
│   ├── LoginPage.tsx         # Simple button that routes to /dashboard
│   └── DashboardPage.tsx     # Placeholder content
├── services/
│   ├── nostr.ts              # Empty - Backend will add applesauce integration here
│   └── api.ts                # Empty - Backend will add API calls here
└── App.tsx                   # Basic routing

LOGIN PAGE (/):
- Centered "Sign in with Nostr" button
- onClick -> navigate to /dashboard
- No actual functionality (just routing)

DASHBOARD PAGE (/dashboard):
- "Dashboard" heading
- Placeholder text: "Waiting for backend integration"
- "Logout" button that routes back to /
